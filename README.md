# 🧭 HEXAGON Navigation App

**Professional Cross-Platform Navigation Application**

## ✨ Features
- 🗺️ Multi-Platform Maps (Google Maps & Earth)
- 🛰️ Satellite Positioning (GPS/Location)
- 🔷 Hexagon Point System
- 🔒 Security Hardened
- 💰 AdMob Monetization
- 📱 iOS, Android, Web Support
- 🚀 Automated CI/CD Deployment

## 🚀 Quick Start

```bash
git clone https://github.com/rodoljubzgonjanin-coder/HEXAGON-navigation-app
cd HEXAGON-navigation-app
npm install
```
